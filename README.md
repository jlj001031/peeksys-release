# PeekSys

PeekSys is a Windows tray utility for quickly checking your PC's current hardware status.

After launch, PeekSys stays in the system tray. Hover over or click the tray icon to open a compact popup with CPU, memory, GPU, disk, network, foreground FPS, and display refresh rate information.

Website: [https://www.peeksys.com](https://www.peeksys.com)

## Features

- Runs quietly in the Windows system tray.
- Supports both hover and click trigger modes.
- Compact popup for CPU, memory, GPU, disk, network, and display metrics.
- Top overlay bar with fixed and auto-hide modes.
- Supports multiple GPU, disk, network adapter, and display output instances.
- Shows current, minimum, and maximum values.
- Displays foreground window FPS and real display refresh rate.
- Supports light, dark, and system themes.
- Optional transparent popup effect.
- Supports English, Simplified Chinese, Japanese, Korean, and German.
- Supports 200 ms, 500 ms, 1 s, and 2 s collection intervals.
- Supports startup on boot.
- Built-in driver status check and repair entry.

## Download

Download the latest installer from the Releases page.

After installation, PeekSys will appear in the Windows system tray. Right-click the tray icon to change the theme, language, trigger mode, top overlay, visible groups, visible columns, collection interval, and startup behavior.

## System Requirements

- Windows 10 or later.
- x64 device.
- Some hardware sensors require administrator permission and driver support.

## Notes

If some metrics are displayed as `--`, the current hardware, driver, or sensor interface may not expose the corresponding data.

The installer is not code-signed yet, so Windows may show a Microsoft Defender SmartScreen warning. Please make sure you download PeekSys only from the official website or the official GitHub Releases page.
